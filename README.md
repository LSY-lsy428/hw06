### 选题名称

### 选题背景与创作思路
本次作业围绕「智能机器人/具身智能」主题，参考“小v讲AI”系列视频的科普风格，制作一段面向大众的AI科普短视频。创作时以通俗易懂为原则，避免堆砌专业术语，通过生活中的例子和直观对比，帮助观众快速理解核心概念。

### 视频内容概述
视频采用「开头-主体-结尾」三段式结构：
1.  **开头（约30秒）**：以提问或场景引入主题，快速点明视频核心，吸引观众兴趣。
2.  **主体（约2分钟）**：
    - 讲解主题的核心概念，例如具身智能的定义、与传统AI的区别、关键技术（感知/规划/控制）等；
    - 结合具体案例或应用场景进行说明，例如工业机器人、服务机器人、AI大模型赋能机器人等；
    - 补充相关背景知识或发展现状，让内容更完整。
3.  **结尾（约30秒）**：总结核心观点，展望相关技术的未来发展，并可发起简单互动。

### 制作信息
- 时长：约[2-4]分钟
- 格式：MP4
- 制作工具：剪映（或其他剪辑工具）
- 呈现形式：[出镜讲解 / 配音旁白] + 素材/图片/演示
